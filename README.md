# Smartcar Java SDK

Get started at https://developer.smartcar.com

## Installation

### Maven

Add to your POM:

```xml
<dependency>
  <groupId>com.smartcar</groupId>
  <artifactId>sdk</artifactId>
  <version>0.0.1</version>
</dependency>
```

### Gradle

Add to your build file:

```groovy
compile "com.smartcar:sdk:0.0.1"
```
